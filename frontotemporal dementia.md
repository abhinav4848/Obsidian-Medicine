# Cortical [[Dementia]] types
1. Alzheimer's
2. Lewy Body Dementia
3. Frontotemporal lobar degeneration (FTLD): 
	- Frontotemporal dementia (Pick's disease)
	- Progressive non fluent aphasia (chronic progressive aphasia, CPA)
	- Semantic dementia
### Common features of FTLD
- Onset <65yo.
- insidious onset. 
- Has family history
- Memory intact and visuospatial skills preserved. 
- Personality change and social conduct problems
- History of presenting illness- will be from a friend.
# Frontotemporal dementia
1. Sexual disinhibition, 
2. struggle with words, 
3. loss of social interest, 
4. Impulsive behaviour, e.g. gambling addiction 
5. Personality change, 
6. Neglecting personal hygiene.

#mnemonic Pick's disease likes to pick & eat sweet things, get angry, have violent outbursts and this history is from caregiver/ somebody else
*differentiate from [[dementia|NPH]]*

#confusion 
**Alzheimer**: memory changes first
**Pick's**: Personality changes first

**[[dementia#Lewy Body dementia linked|LBD]]**: Cognitive before Movement
**Parkinson**'s: Movement before Cognitive
# Imaging
Focal gyral atrophy with a knife-blade appearance

Rx:
Don't use ACE inhibitors or Memantine.
