# Referral criteria
Refer based on [[dyspepsia]]. 

# Complication
can cause [[cancer- gastric|stomach cancer]]. 
# Treatment
- There is no need to check for H. pylori eradication if symptoms have resolved following test & treat. 
- However, if repeat testing is required then a carbon-13 urea breath test should be used
