# Core ethical principles (Big 4)

|                 |                                                                                                                                                                                                                                      |
| --------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Autonomy        | - Respecting a patient’s **right to make their own decisions**<br>- Requires:<br>    - Capacity<br>    - Informed consent<br>    - Freedom from coercion<br><br>**Examples:** <br>- Refusing treatment<br>- Choosing between options |
| Beneficence     | - Acting in the patient’s **best interests**<br>- Promoting **good outcomes**<br><br>**Examples:**<br>- Giving vaccines<br>- Treating illness<br>- Preventive care                                                                   |
| Non-Maleficence | - **“Do no harm”**<br>- Avoid causing unnecessary harm or risk<br><br>**Examples:**<br>- Avoiding unnecessary procedures<br>- Considering side effects                                                                               |
| Justice         | - **Fair distribution of resources and care**<br>- Treating patients **equitably**<br><br>**Examples:**<br>- Organ allocation<br>- NHS resource prioritisation                                                                       |

# Additional Ethical Principles

|                 |                                                                                                                                                                                                                      |
| --------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Double Effect   | - An action with **both good and harmful effects**<br>- Acceptable if:<br>    - Intention = good<br>    - Harm is not the goal<br><br>**Example:** <br>- Giving opioids for pain relief that may depress respiration |
| Confidentiality | - Keeping patient information **private**<br>- Can be broken if:<br>    - Risk to others<br>    - Legal requirement                                                                                                  |
| Veracity        | - **Truth-telling**<br>- Being honest with patients                                                                                                                                                                  |
| Fidelity        | - **Keeping promises / maintaining trust**<br>- Duty of care                                                                                                                                                         |
