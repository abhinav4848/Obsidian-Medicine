**Scenario:** Antibiotic prescribed despite allergy, pharmacist prevents harm. You want to see if changes reduce future errors. 
- PDSA cycle is a quality improvement tool:
    1. Plan – identify problem & plan intervention
    2. Do – implement change
    3. Study – measure effect
    4. Act – adapt/scale/modify
- You implement change, measure effect, and iterate → classic PDSA.

Whereas **Significant Event Analysis (SEA)**: Reflection on a specific incident to improve practice. Useful for one-off reflection, but doesn’t measure effect over time like PDSA.
