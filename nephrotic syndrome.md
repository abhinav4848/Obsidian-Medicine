# Type
## Minimal Change
- Children
- Happens after NSAID/rifampicin/Hodgkin's/Mono
## Membranous GN
- Adult Caucasian, whites, unspecified ethnicity, age > 40yo
## Focal Segmental
- Adult Africans, Black, American, Hispanic, age young adults
# Clinical Features
Proteinuria (>3g/24hr.)
Hypoalbuminemia (<30g/L), Edema
If only 0.3gm/24 hrs.:Â  [[Pre-eclampsia]]
## Others
- Hyperlipidemia,
- hypercoagulable state -> Medium sized proteins like albumin, transferrin, anti-coagulation factors like Antithrombin 3 can get lost -> Can cause vein thrombosis.
- Liable for infections.
- Normotension (hypertension is rare) in MCD. 
# Investigation
Do Urinalysis.
Best diagnostic test: Biopsy -> indicated only if a trial of steroids has failed. 
# Treatment
Prednisolone 
	Albumin & Furosemide can be used as adjuncts. 
[[cytotoxic drugs|Cyclophosphamide]] is the next step for steroid-resistant cases.
