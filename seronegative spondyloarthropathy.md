# Seronegative arthritis
Features: 
- Lack of Rheumatoid factor & Anti-CCP (seronegative)
- [[hla-b27|HLA-B27]] association. 

Group includes
1. [[Ankylosing spondylitis]]: Anterior [[uveitis]]
2. [[Reactive arthritis]]: Conjunctivitis.
3. [[behcet's disease]] 
4. Arthritis that accompanies [[inflammatory bowel disease|IBD]]
5. [[whipple disease]] 
6. psoriatic arthritis

Based on site commonly affected, they can be divided into predominantly axial or predominantly peripheral.

| axial                                                                                                         | peripheral                                                                        |
| ------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------- |
| - ankylosing spondylitis (radiographic axial spondyloarthritis)<br>- non-radiographic axial spondyloarthritis | - psoriatic arthritis<br>- reactive arthritis<br>- enteropathic spondyloarthritis |

# Investigation
See [[antibodies]].

# Treatments
**NSAID**: Rheumatoid arthritis, Ankylosing spondylitis, Reactive arthritis, [[Gout]].
**Steroid**: Sarcoidosis, Churg-Strauss, Wegener, GCA, Polymyositis, Dermatomyositis, Polymyalgia rheumatica.
