# Medication review
STOPP-START criteria are a structured medication review tool for older adults (usually ≥65) designed to detect inappropriate prescribing.

They recognise two different prescribing errors:

| Type of error                                  | Tool      | Meaning                              |
| ---------------------------------------------- | --------- | ------------------------------------ |
| Giving drugs that shouldn’t be given       | STOPP | Potentially inappropriate medication |
| Failing to give drugs that should be given | START | Potential prescribing omission       |
So during a medication review you ask two questions:

1️⃣ What should I STOP?
- STOPP: Screening Tool of Older Persons’ Potentially inappropriate Prescriptions
2️⃣ What should I START?
- START: Screening Tool to Alert doctors to Right Treatment

This dual structure is why the system is useful — many older tools only focused on bad drugs, not missing beneficial drugs.

### STOPP Criterion
aka potentially inappropriate medications (PIMs).

| STOPP rule                                     | Why                      |
| ---------------------------------------------- | ------------------------ |
| Benzodiazepines in patients prone to falls | ↑ falls, fractures       |
| NSAIDs in heart failure                    | fluid retention          |
| Duplicate drug class (two ACE inhibitors)  | no benefit               |
| Long-term PPIs without indication          | fracture, infection risk |
| Anticholinergics in dementia               | worsens cognition        |
**Look at**: Falls risk, Renal impairment, Dementia, Heart failure, Drug interactions, Duration too long

### START Criterion
aka potential prescribing omissions (PPOs). 

| START rule                                      | Why                  |
| ----------------------------------------------- | -------------------- |
| ACE inhibitor in systolic [[heart failure\|HF]] | mortality benefit    |
| Statin in vascular disease                      | secondary prevention |
| Anticoagulation in AF with risk factors         | stroke prevention    |
| Calcium + vitamin D in osteoporosis             | fracture prevention  |
| Laxative with chronic opioid use                | prevent constipation |
