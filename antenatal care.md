# Routine tests
Blood: 
- Blood group & Rh status
- Antibody screen
- Full blood count
- **HIV**
- **Hepatitis B**
- **Syphilis**
Urine: Asymptomatic bacteriuria
Other: [[down's syndrome]], [[childhood syndromes|Edward syndrome]], patau.
### Bacteriuria
If a woman has had asymptomatic bacteriuria in pregnancy, a repeat MSU is required at all subsequent antenatal appointments.
- Prophylactic antibiotics in labour and delivery are only needed if a ==group B streptococcus== is isolated. The antenatal care service will need to be informed so that they can arrange prophylactic antibiotics for labour and delivery.
- A woman does not need to be on prophylactic antibiotics until delivery if she is found to have asymptomatic bacteriuria. She may, however, be treated with a short course of antibiotics if an organism is isolated.
- Important because it can cause increased risk of [[pyelonephritis]], preterm birth. 
- No need for a renal USS. 
- See about [[trimethoprim]] & [[nitrofurantoin]]
