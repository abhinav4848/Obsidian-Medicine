| Eye Condition                                                   | Symptoms                                                                                                                                                                                                                                                                                            |
| --------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [[glaucoma- acute angle closure\|Acute angle closure glaucoma]] | Severe pain (may be ocular or headache),<br>decreased visual acuity, patient sees ==halo== due to corneal edema<br>semi-dilated pupil,<br>hazy cornea                                                                                                                                               |
| Anterior [[uveitis]]                                            | Acute onset,<br>pain,<br>blurred vision and photophobia,<br>small, fixed oval pupil, ciliary flush                                                                                                                                                                                                  |
| [[episcleritis\|scleritis]]                                     | Severe pain (may be worse on movement) and tenderness,<br>may be underlying autoimmune disease (e.g., rheumatoid arthritis).<br>Episcleritis is painless.                                                                                                                                           |
| [[conjunctivitis]]                                              | Purulent discharge if bacterial,<br>clear discharge if viral                                                                                                                                                                                                                                        |
| Subconjunctival haemorrhage                                     | History of trauma or coughing bouts. Check BP as that might be a cause. Say 2-4 weeks to recover & come back if worsening or vision changes.<br>If taking warfarin, check INR. <br>Ask h/o trauma to rule out ICB, orbital roof fracture, black eye. If traumatic cause, consider ophthal referral. |
| [[endophthalmitis]]                                             | Typically red eye,<br>pain and visual loss following intraocular surgery                                                                                                                                                                                                                            |
| [[ophthalmia neonatorum]]                                       | Chlamydia/Gonorrhoea in <4weeks old- same day ophthal refer                                                                                                                                                                                                                                         |
| [[keratitis]]                                                   | Bacterial: common with contact lens wearers. Red eye, photophobia, FB sensation. White spot on pen light. <br><br>Viral: as above, but watery discharge. Faint branching opacity on pen light                                                                                                       |
