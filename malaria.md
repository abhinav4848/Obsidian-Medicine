h/o [[travel infections|travel]] to Africa + meningitis like features + anaemia -> Cerebral malaria.

If Schuffner's dots #named -> latent hypnozoites in liver -> Primaquine (targets all stages)
For non-falciparum, non-hypnozoite malaria -> Chloroquine > [[quinine]] (target active stage)

## Pregnancy
Primaquine is [[drugs contraindicated in pregnancy|c/i in pregnancy]] & breastfeeding
Chloroquine & Proguanil safe in pregnancy but no longer effective in chloroquine-resistant areas.
Mefloquine is now the DOC for Pregnants as prophylaxis in T2&T3 when travelling to chloroquine resistant areas.
