| Drug Class                         | Adverse Effects / Key Points                                                                                                                                                                                                                                     |
| ---------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **[[ACE inhibitors]]**             | Cough; Angioedema; Contraindicated in renal artery stenosis; [[Hyperkalemia]] (due to ↓ aldosterone)                                                                                                                                                             |
| **Beta-blockers (BB)**             | Asthma (bronchospasm risk)<br>fatigue<br>cold peripheries                                                                                                                                                                                                        |
| **Calcium channel blockers (CCB)** | Edema; Gingival hypertrophy                                                                                                                                                                                                                                      |
| **Loop diuretics**                 | HypoNaBpK; HyperU ( #mnemonic)                                                                                                                                                                                                                                   |
| **Thiazide diuretics**             | HypoNaBpK ([[Hyponatremia]], Hypotension, [[hypokalemia]], Mg+)<br>HyperGLUC (glycemia, lipidaemia, [[gout\|uricaemia]], [[hypercalcemia\|calcemia]]) #mnemonic <br>Metabolic Alkalosis<br>Impotence<br><br>#mnemonic Thiazide Takes calcium, Loops Lose calcium |
| **K⁺-sparing diuretics**           | Hyponatremia; Hyperkalemia; [[gynaecomastia]]                                                                                                                                                                                                                    |
