# Management
Try NSAIDs as first line for back pain. Still limited evidence of NSAID benefit for sciatica. 
Ibuprofen up to 1200mg daily. 
Naproxen up to 1000mg daily (NB use plain tablets)

# Red Flags
Always look for red flags:
See this NICE CKS [Link](https://cks.nice.org.uk/topics/sciatica-lumbar-radiculopathy/diagnosis/red-flag-symptoms-signs/)
- CES
- Spinal fracture
- Cancer
- Infection (discitis, vertebral osteomyelitis, spinal epidural abscess)
